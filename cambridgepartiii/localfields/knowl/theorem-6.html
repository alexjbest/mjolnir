Theorem4.2.3
\(\cO_K\) is a Dedekind domain, \(K = \Frac\cO_K\), \(L/K\) finite field extension, \(\cO_L\) integral closure of \(\cO_K\) in \(L\). Let \(p \subset \cO_K\) be a prime ideal, then \(p \cO_L = P_1^{e_1} \cdots P_r^{e_r}\) for distinct primes \(P_i\). Then the absolute values on \(L\) extending \(\ab_p\) on \(K\) are (up to equivalence) \(\ab_{P_1},\ldots,\ab_{P_r}\).
Proof
\(\cO_L\) is Dedekind . Take \(x \in K^*\) we have \(v_{P_i}(x) = e_i v_p(x)\) so \(\ab_{P_i}\) is equivalent to an absolute value extending \(\ab_p\). Now let \(\ab\) be any such absolute value on \(L\), it will be non-archimidean by . \(\cO_K \subset \{x\in L : |x\ \le 1\}\) so taking integral closures in \(L\) we get \(\cO_L \subset \{x\in L : |x| \le 1\}\). Lemma 3.4 gives that \(\ab\sim \ab_P\) for some prime \(P\subset \cO_L\). But \(\ab\) extends \(\ab_p\) so \(P \cap \cO_K = p\) giving \(P = P_i\) for some \(i\).
