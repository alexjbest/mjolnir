              Definition8.0.11Herbrand quotients
              The Herbrand quotient is \(q(A) = |H^0(A)|/|H^1(A)|\), it is undefined if either group is infinite.
              Theorem8.0.12Hilbert's theorem 90
              \(L/K\) Galois with \(\langle \sigma \rangle = G = \Gal(L/K)\) cyclic of order \(n\). If \(x\in L\) with \(N_{L/K} (x) = 1\) then there is \(y \in L^*\) s.t. \(x = \sigma(y)/y\).
              Proof
              Let \(a_0 = 1\), \(a_r = \prod_{i=0}^{r-1} \sigma^i(x)\). As distinct automorphisms are linearly independent there is some \(z\in L\) with \(b=\sum_{i=0}^{n-1} a_i \sigma^i(z) \ne 0\). Then \[\sigma(b) = \sum_{i=0}^{n-1}\sigma(a_i)\sigma^{i+1}(z) = \sum_{i=0}^{n-1}\frac{a_{i+1}}{x}\sigma^{i+1}(z) =\frac{1}{x}\sum_{i=0}^{n-1}a_i\sigma^{i}(z)\]. So \(x\sigma(b) = b\), giving \(x = \sigma(b^{-1})/b^{-1}\).
