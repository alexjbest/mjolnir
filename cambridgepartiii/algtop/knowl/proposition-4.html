Proposition1.4.21
\([a_n]\) generates \(\tilde H_n(S^n) = \ZZ\) (statement \((A_n)\)).
\([b_n]\) generates \(\tilde H_n(S^n, *) = \ZZ\) (statement \((B_n)\)).
Proof
Statement \((A_0)\): \(S^0 = \{p_+, p_-\}\), \(de = e_{p_+} - e_{p_-}\) so \(de\) generates \(\tilde H_0(S^0)\).
We'll show that \(A_{n-1}\implies B_n\). Here \(\partial\colon H_n(S^n, *)=H_n(D^n, S^{n-1}) \xrightarrow{\sim} \tilde H_{n-1}(S^{n-1})\) (by 1.4.18). So it suffices to check that \(\partial [b_n] = [a_{n-1}]\)
